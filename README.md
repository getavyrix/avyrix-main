# Avyrix Validation

## Overview
This repository currently contains a static idea-validation landing page and demo experience focused on the problem of helping developers understand common security vulnerabilities in simpler terms. The goal of the current project is to test whether this concept resonates with developers before expanding it into anything larger.

## Demo Purpose
This project demonstrates the idea of helping developers understand security vulnerability patterns through simplified explanations, example snippets, and guided learning content. It is a validation MVP / concept demonstration, not a production-grade security tool.

Live link: https://arzkhudia09-jpg.github.io/Avyrix-Validation/

## Features
- Static landing page for the product concept
- Interactive demo page for exploring example vulnerability patterns
- Example code snippets for common vulnerability themes
- Simplified educational explanations and fix guidance
- Waitlist page for collecting early interest
- Basic client-side UI behavior and form validation

## Local Setup
1. Open the repository folder in a browser, or run a simple static server from the project root.
2. A typical option is:
   python -m http.server 8000
3. Then open http://localhost:8000/ in your browser.
4. Use the main page for the concept overview, or open demo.html to try the interactive example flow.

## Project Structure
